# CloudDrive-Rapid-Transfer
云盘秒传工具脚本（支持百度/夸克/天翼/123/光鸭网盘），解决无法获取 MD5、页面卡死等问题的稳定修复版。

# ⚡ 云盘秒传工具（修复完整版）

支持在中国主流云盘（百度、夸克、天翼、123 云盘、光鸭）之间进行文件秒传和元数据（JSON）导出的油猴脚本。

## ✨ 核心功能
- 支持百度网盘、123 云盘、夸克网盘、天翼云盘一键生成秒传 JSON。
- 支持向光鸭网盘一键批量导入 JSON 并秒传。
- 修复了大量原版 Bug（性能防抖优化、123 云盘虚拟列表抓取修复、百度网盘 MD5 解密修复等）。
- 新增天翼云盘家庭盘页面的兼容适配（实验性支持）。
- 新增调试日志模式，便于定位按钮不显示、选中项识别失败、目录遍历失败等问题。

## ✅ 平台支持矩阵
- 百度网盘个人页：支持
- 百度网盘分享页：不支持（需先保存到我的网盘）
- 夸克个人页：支持
- 夸克分享页：支持（需要手动提供 Cookie）
- 天翼云盘个人页：支持
- 天翼云盘分享页：支持 / 视页面结构可能有差异
- 天翼云盘家庭盘页：实验性支持
- 123 云盘个人页：支持
- 123 云盘分享页：不支持（需先保存到我的网盘）
- 光鸭网盘 JSON 导入：支持

## 🧪 天翼家庭盘说明
目前已对以下家庭盘页面做兼容增强：
- `https://cloud.189.cn/web/family/file/folder/home`

当前家庭盘支持包括：
- 页面识别与按钮挂载
- 选中文件读取（unsafeWindow / React fallback / DOM-Vue fallback）
- 目录递归遍历时的家庭盘参数兼容
- 文件详情 / MD5 补全时的家庭盘参数兼容

注意：
- 家庭盘仍属于**实验性支持**。
- 若你遇到“按钮已出现但导出为空”“选中文件夹无结果”等情况，请开启调试模式并提供控制台日志反馈。

## 🐞 调试模式（重要）
当你遇到下面这些问题时，建议先开启调试：
- 按钮不显示
- 点“生成秒传 JSON”没有反应
- 天翼家庭盘导出为空
- 已勾选文件，但脚本提示未勾选
- 文件夹递归没有结果

### 开启方式（推荐）
安装脚本后，在油猴脚本菜单里直接使用：
- `开启调试日志`
- `关闭调试日志`

### 手动开启方式
在浏览器控制台执行：

```js
localStorage.setItem("guangya_debug", "1")
location.reload()
```

关闭：

```js
localStorage.removeItem("guangya_debug")
location.reload()
```

### 调试模式会输出什么
控制台会输出类似前缀：

```text
[秒传工具][debug]
```

重点包括：
- 当前页面识别结果
- 当前页面是否被识别为主页面 / 分享页 / 家庭盘
- 读取选中项时使用了哪条路径
  - unsafeWindow
  - React fallback
  - DOM / Vue fallback
- 选中项数量和部分样本
- 天翼目录遍历的分页情况
- API 返回非 0 `res_code` 的告警
- 最终收集到的文件数量

### 反馈问题时建议提供
如果你要提 issue 或找我继续修，建议一起提供：
1. 使用的是哪个网盘页面（个人页 / 分享页 / 家庭盘）
2. 具体 URL（可打码）
3. 是否勾选了文件 / 文件夹
4. 浏览器控制台中的 `[秒传工具][debug]` 日志
5. 如果报错，附上完整报错文本截图或复制文本

## ⚠️ 注意事项
- 夸克网盘分享页提取需要手动抓取并填入 Cookie。
- 百度网盘和 123 云盘在分享页无法提取，**必须先保存到“我的网盘”**，在个人主页操作提取。
- 导入光鸭网盘时，如果提示“秒传失败”，说明光鸭服务器暂未收录该文件，需手动上传一次。
- 某些文件如果页面或接口拿不到 MD5，可能会被跳过（取决于当前 etag 策略）。
- 天翼家庭盘支持仍在持续完善中，欢迎反馈页面结构差异。

## ❓常见问题
### 1. 按钮不显示怎么办？
- 先刷新页面。
- 确认当前页面属于支持范围。
- 开启调试模式，看是否识别到了正确页面类型和挂载点。

### 2. 明明勾选了文件，为什么提示“请先勾选文件”？
- 可能是页面升级后选中项结构变了。
- 先开 debug，看脚本是在哪一层（unsafeWindow / React / DOM）失败。

### 3. 天翼家庭盘按钮出来了，但导出为空怎么办？
- 这是当前重点排查场景。
- 请开启 debug，并把控制台日志贴出来，尤其是：
  - 页面识别
  - 选中项来源
  - listPersonal 分页日志
  - collected files 数量

### 4. 为什么有些文件没有导出成功？
- 可能是页面没有返回 md5 / etag。
- 也可能是接口详情补查失败。
- 调试模式会显示这类过程信息。
