# ⚡ 云盘秒传工具 (修复完整版)

支持在中国主流云盘（百度、夸克、天翼、123云盘、光鸭）之间进行文件秒传和元数据（JSON）导出的油猴脚本。

## ✨ 核心功能
* 支持百度网盘、123云盘、夸克网盘、天翼云盘一键生成秒传 JSON。
* 支持向光鸭网盘一键批量导入 JSON 并秒传。
* 修复了大量原版 Bug（性能防抖优化、123云盘虚拟列表抓取修复、百度网盘 MD5 解密修复等）。

## ⚠️ 注意事项
* 夸克网盘分享页提取需要手动抓取并填入 Cookie。
* 百度网盘和 123 云盘在分享页无法提取，**必须先保存到“我的网盘”**，在个人主页操作提取。
* 导入光鸭网盘时，如果提示“秒传失败”，说明光鸭服务器暂未收录该文件，需手动上传一次。
